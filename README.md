🎬 Disney+ Data Analysis Dashboard

An interactive Disney+ Content Analysis Dashboard built using Python to explore and visualize the Disney+ dataset. This dashboard provides meaningful insights into content distribution, genres, release trends, ratings, and country-wise availability through interactive visualizations.

📌 Project Overview

The goal of this project is to perform Exploratory Data Analysis (EDA) on the Disney+ dataset and present key insights using an interactive dashboard. It helps users understand the platform's content library by analyzing movies, series, genres, release years, content ratings, and country distribution.

📈 Key Insights

Disney+ library is dominated by Movies (76.1%), followed by Series (21.4%) and Episodes (2.5%).
The United States contributes the highest amount of content.
Family and Comedy are the most popular genres.
Content additions have increased significantly after the year 2000, with a peak around 2019–2020.
The most common content rating is G, indicating a strong focus on family-friendly entertainment.
The dataset contains 894 titles with an average IMDb rating of 6.66.
